Aggregator

This iPhone app project can download and parse RSS 1.0, RSS 2.0 and Atom web feeds.

The following web feeds are used to demonstrate how application parses the given web feed to display title of a feed, list all of the feed items, and display an item in more detail when tapped. When user clicks on individual feed item in detail view, application will open original website content in browser, with current state of application preserved.

  • Top Stories from CNN.com : http://rss.cnn.com/rss/cnn_topstories.rss
  • ESPN.com sports news: http://sports.espn.go.com/espn/rss/news
  • A multilingual technical weblog Engadget.com: http://www.engadget.com/rss.xml

The above three are among the most popular and most frequently updated rss feeds online. So I used them for demonstration purpose.

Feel free to pull the source and make changes :-)
